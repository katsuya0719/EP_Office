How to use this application

This application can help energy simulation user to reduce the tedious work for post processing and dig into the data analysis more. You can
  • EnergyPlus User
  • OpenStudio User
  • IES VE User

Step1. Set idf file before running simulation

Once you set all the setting for energy simulation, set following setting.
  1. Set Output:Table:SummaryReports to AllSummary

  2. In OutputControl:Table:Style chapter, set Column Separator to CommaAndHTML and Unit Conversion to JtoKWH

Step2. After finished simulation, go to upload page and input all the information and upload html file from the result file
